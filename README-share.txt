YTDL

Windows quick start
Windows 快速開始

1. Unzip the entire YTDL-WIN.zip folder.
   完整解壓縮 YTDL-WIN.zip。
2. Do not run files directly inside the ZIP preview window.
   請勿直接在 ZIP 預覽視窗內執行檔案。
3. Run 01.Install.bat once to create a desktop shortcut.
   第一次使用時執行一次 01.Install.bat，建立桌面捷徑。
4. Later, use the desktop shortcut or 02.RUN.bat.
   之後可使用桌面捷徑或 02.RUN.bat 啟動。

Notes
注意事項

- This package includes its own Python runtime and required components.
  此版本已包含獨立的 Python 執行環境與必要元件。
- You do not need to install Python separately.
  不需要另外安裝 Python。
- Keep the whole extracted folder together.
  請保留完整的解壓縮資料夾，不要單獨移動內部檔案。
- Only download content you have the right to save or use.
  請只下載您有權儲存或使用的內容。
