YTDL

請先完整解壓縮 ZIP，不要在 ZIP 視窗裡直接執行。

第一次使用：
1. 在 YTDL-GUI.zip 上按右鍵，選擇「解壓縮全部」或用 7-Zip 解壓縮。
2. 打開解壓後的 YTDL 資料夾。
3. 雙擊 01.Install.bat 建立桌面捷徑。
4. 之後從桌面捷徑或 02.RUN.bat 啟動。

一般使用：
- 雙擊桌面的 YTDL 捷徑。
- 或雙擊 02.RUN.bat。

備註：
- ZIP 已包含 Python、Python 套件、Deno、Flet runtime。
- 新電腦不需要先安裝 Python。
- 第一次使用也不需要 pip 安裝 Python 套件。
- 請不要刪除 app 資料夾。
- 請只下載你有權保存或使用的內容。

原始碼與更新：
- 專案以開源方式維護；可攜版請從 GitHub Releases 下載最新 ZIP。
